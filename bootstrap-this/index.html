What is Bootstrap?

Bootstrap is a free programme that can be downloaded and used by anyone. It is a toolkit to help create responsive websites. With Bootstrap, a web developer can use the grid system to create a complex layout very fast.
If you want to build a responsive, mobile-first site, Bootstrap is the go-to framework. It is an open source toolkit for developing with HTML, CSS, and JS.

Features of Bootstrap

Responsiveness

Bootstrap uses a 12-grid system for creating mobile-ready websites. It uses two patterns: the grid and the flex systems.

Reboot

The CSS reset style-sheet for bootstrap is Reboot.

Responsive Breakpoint

Bootstrap makes it easy to build responsive layouts due to the predefined classes which cover extra-small, small, medium, large, and extra-large screen widths.

Utilities

This is what makes CSS styling with Bootstrap possible. The main component to be utilized for customization in Bootstrap is CSS. Examples of the utilities are:

  • Flex
  • Color
  • Margin
  • Padding

How to Use Bootstrap

Here are the steps to be taken to start using Bootstrap:

  • Go to https://getbootstrap.com/
  • Click on ‘Get Started’
  • Scroll down the page to copy the Starter Template
  • Paste the template on Visual Studio Code

If all the steps above are done correctly, you can start styling your html using Bootstrap’s utilities.
