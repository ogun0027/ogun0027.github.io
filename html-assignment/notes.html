5 HTML Tags I Have Learnt

Here are the 5 HTML Tags that I have learnt and their descriptions.
< p>
This is called a paragraph tag and it indicates that the text in it is a paragraph text.
< ol>
This stands for ordered list. It provides structure.
< ul>
This tag stands for unordered list.
< body>
This tag indicates a section of html that contains content.
< img>
This is an image tag. It is used to add an image on a page.
