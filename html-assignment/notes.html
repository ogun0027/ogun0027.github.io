5 HTML Tags I Have Learnt

Here are the 5 HTML Tags that I have learnt and their description.
  • < p> - This is called a paragraph tag and it indicates that the text in it is a paragraph text.
  • < ol> - This stands for ordered list. It provides structure.
  • < ul> - This tag stands for unordered list.
  • < body> - This tag indicates a section of html that contains content
  • < img> - This tag indicates that an image will be shown on the page
